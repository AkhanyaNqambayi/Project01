# To-Do App

A simple JavaScript To-Do app that lets you manage your daily tasks. Add new tasks, mark them as completed, delete tasks, and keep your list saved even after closing the browser using localStorage.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage with localStorage
- Simple and user-friendly interface
